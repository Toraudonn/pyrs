# pyrs
*another* Python wrapper for Intel Realsense camera


## Dependencies:

- OpenCV (cv2 for Python)
- librealsense2 (pyrealsense2)
- numpy

## Usage:


To start the pyrs demo, run:

```
python pyrs.py
```

### Commands:

- `p`: save rgb and depth image
- `c`: change camera presets
- `i`: save camera intrinsics
- `q`: quit loop
